Анализ макроэкономических данных
Вариативности нет, составляет данные для столбца GDP

Код читает все таблицы данных и выводит таблицу [  city /   GDP  ]
city - название города
GDP - ВВП страны (среднее арифметическое значение из всех таблиц если идет учет на несколько лет)

Читает файлы формата .csv в текущей дериктории.
Необходимо просто добавить новый файл в текущую папку проекта

Запуск через teminal
пример: python main.py --files economic1.csv economic2.csv --report average-gdp
        python main.py --files economic1.csv economic2.csv
